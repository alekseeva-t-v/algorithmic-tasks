Самый частый элемент

Условия задачи:

Найти, сколько раз встречается самый частый элемент в объединении двух отсортированных по возрастанию массивов. Элементы могут повторяться.

Примеры:

countMostFrequent([1, 2, 2, 3], [0, 2, 4, 4]) => 3

countMostFrequent([], [0, 0]) => 2

Результат:

Сложность решения:

O(n + m)
