Задание с числами в тесте IQ

Условия задачи:

Написать функцию для прохождения типового задания с числами в тесте iq — из списка чисел найти то, которое отличается по чётности от остальных, и вернуть его позицию.

Примеры:

iqTest("2 4 7 8 10") => 3

iqTest("1 2 1 1") => 2

Результат:

Сложность решения:

O(n)
